# OpsPilot AI — Backend

FastAPI backend for [OpsPilot AI](https://github.com/omrzk/opspilot-ai): log parsing and
normalization (EVTX, Sysmon, Defender, Azure, VMware, CloudTrail, Kubernetes, syslog),
AI root-cause analysis with remediation scripting, RAG over pgvector, report generation,
and Celery workers.

See the [repository README](../README.md) and [docs/](../docs) for setup, configuration
and architecture.
